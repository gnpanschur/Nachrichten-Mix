Bedienungsanleitung

Recherche (div. URLs)

Startet den automatischen Recherche-Prozess. Das System ruft vordefinierte Nachrichtenquellen (URLs) ab, extrahiert relevante Artikel und bereitet diese für die Weiterverarbeitung vor.

  • URLs in das Textfeld eingeben: (Kann auch aus einer von dir vorgefertigten Liste kopiert werden)
  • Start Recherche: Beginnt den Prozess. Status und Logs werden im schwarzen Fenster angezeigt.
  • Upload Skript: Lädt die verarbeiteten Nachrichten auf den Server (GitHub) hoch (integriert in Recherche-Log).

Video2Text

Lädt YouTube-Videos herunter, transkribiert den Ton in Text und übersetzt diesen optional ins Deutsche.

  • YouTube URL: Hier den Link zum Video einfügen.
  • Translate to German: Haken setzen, um den Text automatisch ins Deutsche zu übersetzen.
  • Start VideoDownload: Lädt das EINE Video der URL herunter und verarbeitet es.
  • Transcribe All Downloads: Verarbeitet ALLE Videos, die sich bereits im "Downloads"- Ordner befinden und erzeugt daraus Text-Dateien.
  • Downloads: Öffnet den Downloads-Ordner der Videos und Texte im Windows-Explorer. Diese Texte kannst du dann in Text To JSON Konverter einfügen.

Text To JSON Konverter

Tool zur manuellen oder automatischen Umwandlung von Texten in das System-Format.

  • Textfeld: Text eingeben, reinkopieren, oder .txt hineinziehen.
  • Einstellungen: Länge der Zusammenfassungen oder "Single Headline" Modus wählen. (Mehrere Texte haben automatisch mehrere Headlines).
  • Analysieren: KI generiert JSON-Output.
  • Speichern: Über das Disketten-Symbol direkt in die App speichern.

Systemsteuerung

  • Zur App: Öffnet die Nachrichten-App in einem neuen Tab.
  • App/data: Öffnet den Datenordner der App im Explorer.
  • Alles Schließen: Fährt das gesamte System herunter (Server, Terminals, Prozesse). Vorsicht!
